# BMI-Calculator-python
A Simple BMI Calculator built using Python that  Calculator Body Mass Index based on Height and weight.
This is a Simple BMI Calculator built using python.
## Features 
.Calculator Body Mass Index (BMI)
.Show BMI category (Underweight,Normalweight,Overweight,Obese)

## HOW TO RUN
1. DOWNLOAD THE FILE
2. OPEN TERMINAL
3. RUN THE COMMAND
PYTHON BMI_CALCULATOR.PY
